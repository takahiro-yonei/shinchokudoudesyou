Connector for Sencha Touch Apps to Interface with Salesforce REST API


This package contains a proxy class that allows Sencha Touch stores to interact with the Salesforce backend.

  • Check out the guide to get started.
  • Read the Ext.force.Force documentaton for how to authenticate to Salesforce and use the low level API to fetch information.
  • Read the Ext.force.ForceProxy documentaton for instructions on how to connect a Sencha Touch store to Salesforce.
